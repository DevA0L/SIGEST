# Sistema de Gerenciamento de Estágio Técnico
Sistema voltado para a automatização de todo o processo de estágio. O usuário pode escolher como deseja acessar o site, se é como aluno, instituição ou empresa, dependendo da sua escolha o mesmo terá  acesso a diferentes telas do site.

**`Alunos`** poderão acessar documentos necessários para a realização do seu estágio, terá um perfil para personalizar da maneira que desejar, terá acesso a candidaturas e vagas e poderá acompanhar o proresso do seu estágio.

**`Instituição`** irá controlar todo o processo de estágio, desde as vagas ofertadas até a documentação do aluno.

**`Empresa`** terá acesso as vagas, canddidaturas e informações do perfil do aluno.
